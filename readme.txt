== Description ==

Gives you ability to add tabs to posts, pages, & text widget

See <a href="http://mortgageloaninsight.com" target="_blank">mortgage loan insight</a> to get more info

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `mortabs.zip` to the `/wp-content/plugins/` and extract it.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Save the settings in settings -> Mortabs.
4. Test and enjoy!

= Usage =
Paste the shortcode on your posts or pages


{tab=Tab 1}

Contents Tab 1

{tab=Tab 2}

Contents Tab 2

{/tabs}
