== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `tabs-extended.zip` to the `/wp-content/plugins/` and extract it.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Save the settings in settings -> Tabs Extended.
4. Test and enjoy!
